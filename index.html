Timestamp Microservice

Free Code Camp API Project

What is does

I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp (example: 1451628000) or a natural language date (example: January 1, 2016).

If it does, it returns both the Unix timestamp and the natural language form of that date.

If it does not contain a date or Unix timestamp, it returns null for those properties.

https://time-stampapi.herokuapp.com/November 18, 1979

will return

{ unix: "311731200", natural: "November 18, 1979" }

and

https://time-stampapi.herokuapp.com/1509426000

will return

{ unix: "1509426000", natural: "October 31, 2017" }
